reporting bugs & request features

if you encounter an issue or want to suggest new options, please open a ticket on our github issue tracker:

open an issue on github

when reporting a bug, please include:

  • the chrome version you are using.
  • a screenshot of the target website's console logs showing the cors/csp blocking error.
  • your extension configuration settings.
